前端基础

hojun · 2019-7-15 · 次阅读

HTML

块级元素、行内元素

  • 块级元素会自动跳行，行内不会；
<div> 属于块级元素 用作元素的容器，常用来网页布局
<span> 属于内联元素 用作文本的容器

块级标签

标签 意义 标签 意义
center 居中对齐块 dir 目录列表
div 常用块级元素 dl 定义列表
form 交互表单 h1-h6 标题
hr 水平分割线 menu 菜单列表
ol 排序表单 p 段落
address 地址 blockquote 块应用

CSS

选择器

selector{dec} //选择标签名
#id{dec} //id选择器
.ClassName{dec} //选择类名
[att]{dec} //属性选择器

selector1 selector2 {dec}//派生选择器
#id selector{dec} // 选择标签名与id都满足的

引入

  • link引入外部css
  • 内部style标签
  • 标签内嵌style属性

继承与覆盖

  • 即颜色属性将被继承于外部样式表，而文字排列（text-alignment）和字体尺寸（font-size）会被内部样式表中的规则取代。

布局相关属性

  • 盒子模型：每一个标签都是一个盒子
    padding 内边距
  • 背景属性是作用于整个盒子，包括内边距那部分
postion 把元素放置到一个静态的、相对的、绝对的、或固定的位置中。
float 定义元素在哪个方向浮动。
visibility 设置元素是否可见或不可见。
cursor 规定当指向某元素之上时显示的指针类型。
clear 设置一个元素的侧面是否允许其他的浮动元素。
display 设置是否及如何显示元素。

padding 内边距

  padding-top: 10px;
  padding-right: 0.25em;
  padding-bottom: 2ex;
  padding-left: 20%;

margin 外边距

  • 与其他盒子的距离
  margin-top: 20px;
  margin-right: 30px;
  margin-bottom: 30px;
  margin-left: 20px;

border

  • 边框的样式
  • 边框的颜色
  • 边框的宽度

边距合并

  • 上下元素合并
  • 里外元素合并

position

position:relative;
left:20px //相对于原始位置向左移动

position:absolute
left:100px;
top:150px //在整个页面中的定位

position:fixed;
top:150px //在整个页面中的定位，不会随着滚动条移动

float

  • 将边框向个方向移动直到遇见其他边框
.center
{
margin-left:auto;
margin-right:auto;
width:70%;
background-color:#b0e0e6;
}//使用margin水平对齐

.left
{
position:absolute;
left:0px;
width:300px;
background-color:#b0e0e6;
}//使用position左对齐

.left
{
float:left;
width:300px;
background-color:#b0e0e6;
}//使用float左对齐

js

给未声明的变量赋值时会自动成为全局变量，即使是在函数内部声明的

this关键词

  • 在方法中(对象中的函数)，this 指的是所有者对象。
  • 单独的情况下，this 指的是全局对象。
  • 在函数中，this 指的是全局对象。
  • 在函数中，严格模式下，this 是 undefined。
  • 在事件中，this 指的是接收事件的元素。

DOM 文档对象模型

利用document对象去修改HTML文档的所有内容

BOM浏览器对象模型

使用window对象执行与浏览器相关的操作， Window 对象会在 body 或 frameset标签每次出现时被自动创建。

var c = a || b// 如果a的值为True就将a赋值给c，反之将b赋值给c
var c = a && b //如果a的值为True就将b赋值给c，反之将a赋值给c
doucment 文档对象
let 变量修改词，增加变量的块作用域

JQuery

类似于js中的DOM操作，但是语法更加简单，是一个封装的js库

基础语法 $(selector).action()

  • 美元符号定义 jQuery
  • 选择符（selector）”查询”和”查找” HTML 元素
  • jQuery 的 action() 执行对元素的操作
代码 解释
$(“#test”) 选择id
$(“.class”) 选择类
$(“[attribute]”) 选择某一个属性
$(“div”) 选择标签

JQuery 事件

当某一个元素发生制定时间时，执行的函数

$(“button”).click(function(){})

JQuery与AJAX

$(selector).load(URL,data,callback);可以将服务器中加载的数据放入标签中

//$.get(URL,callback);
$("button").click(function(){
  $.get("demo_test.php",function(data,status){
    alert("数据: " + data + "\n状态: " + status);
  });
});

//$.post(URL,data,callback);
$("button").click(function(){
    $.post("/try/ajax/demo_test_post.php",
    {
        name:"菜鸟教程",
        url:"http://www.runoob.com"
    },
        function(data,status){
        alert("数据: \n" + data + "\n状态: " + status);
    });
});
